# gh-proxy-go

这是 [gh-proxy](https://github.com/hunshcn/gh-proxy) 的 Go 语言版本。

## 部署详情

[github.moeyy.xyz](https://github.moeyy.xyz/) 正在使用 **gh-proxy-go**，托管在 [BuyVM](https://buyvm.net/) 每月 3.5 美元的 1 核 1G 内存、10Gbps 带宽服务器上。

### 服务器概况：

- **日流量处理**：约 3TB
- **CPU 平均使用率**：20%
- **带宽平均占用**：400Mbps
